# 1. React Native Foundation (For React JS Developers)

## What changes from React JS to React Native

- You still build UI with components and hooks.
- There is no HTML or CSS file.
- You use native primitives from react-native:
  - View (like div)
  - Text (for all visible text)
  - Pressable (tap/click interactions)
  - ScrollView (scrollable container)
- Styling is JavaScript objects through StyleSheet.create.

## Why Expo is used here

- Expo gives fast setup, dev server, and easy multi-platform testing.
- This project uses expo-router for file-based navigation.
- You can run one codebase on Android, iOS, and web.

## Mental model for this project

- app/\_layout.tsx defines the root navigator (Stack).
- app/(tabs)/\_layout.tsx defines tab navigation.
- app/(tabs)/index.tsx is the reviewer quiz.
- app/(tabs)/explore.tsx is the learning screen.
- app/modal.tsx is a modal glossary screen.

## Core flow you built

1. User selects an answer.
2. User submits answer.
3. App checks correctness.
4. App shows explanation.
5. User goes next question.
6. App tracks score and can restart.
