# 3. Expo Router Routing System

## File-based routing in this app

- app/\_layout.tsx
  - Root Stack for all routes.
- app/(tabs)/\_layout.tsx
  - Tabs navigator for grouped routes.
- app/(tabs)/index.tsx
  - Route: / (first tab screen)
- app/(tabs)/explore.tsx
  - Route: /explore
- app/modal.tsx
  - Route: /modal (shown as modal via Stack options)

## Why \_layout.tsx exists

- \_layout.tsx wraps sibling route files.
- In root: controls global Stack behavior.
- In tabs group: controls tab behavior for files in that folder.

## Stack vs Tabs

- Stack is for push/pop style navigation (screen history).
- Tabs are peer sections for fast switching.
- You can combine them: this project uses Tabs inside Stack.

## How navigation happens here

- Tap tab item -> Tabs switches between index and explore.
- Press Link to /modal -> Stack presents modal screen.
- Link with dismissTo returns and closes modal stack entry.
