# 2. Imports and Components Deep Dive

This file explains every import from Expo or React Native used in the executed app code.

## app/_layout.tsx

- import { DarkTheme, DefaultTheme, ThemeProvider } from '@react-navigation/native'
  - DarkTheme and DefaultTheme are predefined navigation theme objects.
  - ThemeProvider injects the chosen theme to navigation-aware components.
- import { Stack } from 'expo-router'
  - Stack creates screen-to-screen navigation behavior.
- import { StatusBar } from 'expo-status-bar'
  - Controls status bar style in a cross-platform way.

Why it matters:
- This file is your app shell. It decides global navigation and visual theme behavior.

## app/(tabs)/_layout.tsx

- import { Tabs } from 'expo-router'
  - Tabs creates bottom-tab navigation from files in the same folder.

Why it matters:
- This is how Reviewer and Learn tabs appear and switch.

## app/(tabs)/index.tsx

- import { useMemo, useState } from 'react'
  - useState stores mutable state per render cycle.
  - useMemo caches computed text to avoid unnecessary recalculation.
- import { Pressable, SafeAreaView, ScrollView, StyleSheet, Text, View } from 'react-native'
  - SafeAreaView: keeps content away from notches/system bars.
  - ScrollView: allows vertical scrolling if content is long.
  - View: layout container.
  - Text: renders text.
  - Pressable: touchable action element for answers and buttons.
  - StyleSheet: creates organized, validated style objects.

Why it matters:
- These imports are enough to build a full quiz experience without extra UI libraries.

## app/(tabs)/explore.tsx

- import { Link } from 'expo-router'
  - Link navigates to another route using file-based path.
- import { SafeAreaView, ScrollView, StyleSheet, Text, View } from 'react-native'
  - Same purpose as in index.tsx.

Why it matters:
- You teach navigation and structure in-app, not only in docs.

## app/modal.tsx

- import { Link } from 'expo-router'
  - Link with dismissTo returns from modal to a route.
- import { SafeAreaView, ScrollView, StyleSheet, Text, View } from 'react-native'
  - Used to format glossary content in a modal screen.

Why it matters:
- You now have an isolated learning screen that does not clutter quiz logic.

## Component responsibilities in one line

- RootLayout: app shell + Stack + status bar + theme.
- TabLayout: bottom tabs for primary sections.
- ReviewerScreen: stateful quiz engine and UI.
- LearnScreen: in-app explanation of architecture.
- ModalScreen: compact glossary of important imports.
