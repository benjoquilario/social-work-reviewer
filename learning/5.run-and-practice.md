# 5. Run and Practice Plan

## Run commands

1. npm install
2. npm run start
3. Press a for Android, i for iOS, or w for web

## Learning drills

1. Add 3 new questions to QUESTIONS in app/(tabs)/index.tsx.
2. Add a difficulty field to each question (easy, medium, hard).
3. Show difficulty badge in the card.
4. Add a progress bar based on currentIndex / QUESTIONS.length.
5. In learn screen, add links to specific docs in learning folder.

## Suggested next milestone

- Move QUESTIONS into a separate data file and import it.
- Add categories (React hooks, routing, styling, performance).
- Track wrong answers and create a retry-only-incorrect mode.
